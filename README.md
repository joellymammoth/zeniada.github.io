# [Zeniada Literary Magazine](http://www.zeniada.com/)

[New Age](http://startbootstrap.com/template-overviews/new-age/) is a web app landing page theme for [Bootstrap](http://getbootstrap.com/) created by [Start Bootstrap](http://startbootstrap.com/).

## About
Founded in 1929, Zeniada is Johns Hopkins University's oldest and only intercollegiate literary magazine. Every spring and fall, we publish original poetry, prose, and artwork from undergraduates across the country.
